# Japanese Translation Assistant

这是一款chrome插件，旨在帮助用户更方便地将英文文本翻译成日语，并朗读翻译结果。
1. **快速翻译**: 当用户在网页上选中英文文本后，插件会在鼠标附近自动显示对应的日语翻译，无需额外的操作。
2. **朗读功能**: 用户可以点击显示的--日语翻译按钮--，插件将会朗读该翻译内容帮助用户更好地理解和掌握泰语发音。
